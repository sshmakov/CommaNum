CommaNum - программа для работы с номерами в буфере.

I. Общее описание

 

Выделяет из скопированного в буфер произвольного текста номера, может делать некоторые действия над ними

  • формировать список номеров в одной строке через запятую, в столбец
  • запоминать список номеров
  • делать объединение списков
  • находить различие в списках, т.е. выделять номера, которые есть в одном списке, но нет в другом

II. Установка

 1. Запустить инсталлятор CommaNum_setup_версия.exe

2. Инсталлятор установит программу, добавит ее в папку автозагрузки и запустит ее

3. В области уведомлений панели задач (возле часов) появится иконка программы

 

III. Использование

Получение списка номеров

1. Скопировать (Ctrl-C) в буфер текст, содержащий номера. Копировать текст можно из любого места, в том числе из Excel, Word, Project. Таблицы, рисунки, другие "вкрапления" в тексте не мешают

2. Щелкнуть левой кнопкой мышки на иконке в трее - программа преобразует буфер, оставляя только числа, сортирует их, убирает дубликаты и формирует список номеров через запятую.

3. Вставить текст из буфера в нужное место - в Tracker, Word, письмо и т.д.

 

Память номеров

Программа имеет память, как в калькуляторе. В памяти можно запомнить список номеров, затем использовать запомненный список для операций.

  

Меню операций

Чтобы использовать эти операции, нажмите правую клавишу мышки на иконке в панели задач - откроется меню операций.

Операции:

В список номеров

Текст в буфере преобразуется в список номеров, разделенных запятой и пробелом

Пример – текст:

  

LinkedSCR   [28.05.2009 15:32:00(Fradkin)]: 

  

   #142235, #142236 

  

будет преобразован в список номеров: «142235, 142236»

В список номеров с #

Текст в буфере преобразуется в список номеров, разделенных запятой и пробелом, перед каждым номером ставится '#'

Пример – текст «142235, 142236» будет преобразован в текст «#142235, #142236»

В столбец

Текст в буфере преобразуется в список номеров в столбец

Пример – текст «142235, 142236» будет преобразован в таблицу:

142235

142236

Запомнить список номеров

Из текста в буфере выделяется список номеров, сохраняется памяти программы. Содержимое буфера не меняется.

Память + Буфер

Текст в буфере преобразуется в список номеров, к нему добавляется список из памяти, оставляются только уникальные значения.

Память – Буфер

Текст в буфере преобразуется в список номеров, затем сравнивается со списком из памяти, номера, которые есть в памяти, но нет в буфере, записываются в буфер

Память & Буфер

Текст в буфере преобразуется в список номеров, затем сравнивается со списком из памяти, номера, которые есть и в памяти, и в буфере, записываются в буфер

Взять из памяти

Список номеров из памяти копируется в буфер

Очистить память

Очистить память программы. Содержимое буфера не меняется.

Таблица сравнений

Формируется таблица сравнений номеров в памяти и в буфере (Память, Буфер, Совпадающие, Нет в памяти, Нет в буфере)
