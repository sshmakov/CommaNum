CommaNum - программа для работы с номерами запросов в буфере.

Выделяет из скопированного в буфер произвольного текста номера запросов, может делать некоторые действия над ними
- формировать список номеров в одной строке через запятую
- запоминать список номеров
- делать объединение списков
- находить различие в списках, т.е. выделять запросы, которые есть в одном списке, но нет в другом

    I. Установка

1. Запустить инсталлятор CommaNum_setup.exe
2. Инсталлятор установит программу, добавит ее в папку автозагрузки и запустит ее
3. В области уведомлений панели задач (возле часов) появится иконка программы



    II. Использование

  Получение списка номеров запросов

1. Скопировать (Ctrl-C) в буфер текст, содержащий номера запросов. Копировать текст можно из любого места, 
в том числе из Excel, Word, Project. Таблицы, рисунки, другие "вкрапления" в тексте не мешают

2. Щелкнуть левой кнопкой на иконке в трее - программа преобразует буфер, оставляя только числа >= 10000, 
сортирует их, убирает дубликаты и формирует список номеров через запятую.

3. Вставить текст из буфера в нужное место - в Tracker, Word, письмо и т.д.



  Работа со списком запросов в Tracker

Если скопировать список выделенных запросов в Tracker, а затем щелкнуть на иконке, то программа сначала исправляет буфер - преобразует его в текст в правильной кодировке со списком значений столбцов Трекера, разделенных запятыми, который можно вставить в Excel и разнести по столбцам.

  Операции над списком номеров

Программа имеет память, как в калькуляторе. В памяти можно запомнить список номеров, затем использовать 
запомненный список для операций.
Чтобы использовать эти операции, нажмите правую клавишу мышки на иконке в панели задач - откроется меню операций.
Операции:
"В список номеров"          - текст в буфере преобразуется в список номеров, разделенных запятой и пробелом
"В список номеров с #"      - текст в буфере преобразуется в список номеров, разделенных запятой и пробелом, 
                              перед каждым номером ставится '#'
"В столбец"                 - текст в буфере преобразуется в список номеров в столбец
--                          
"Запомнить список номеров"  - из текста в буфере выделяется список номеров, сохраняется памяти программы. 
                              Содержимое буфера не меняется.
"Память + Буфер"            - текст в буфере преобразуется в список номеров, к нему добавляется список из памяти,
                              оставляются только уникальные значения.
"Память - Буфер"            - текст в буфере преобразуется в список номеров, затем сравнивается со списком из памяти,
                              номера, которые есть в памяти, но нет в буфере, записываются в буфер
"Память & Буфер"            - текст в буфере преобразуется в список номеров, затем сравнивается со списком из памяти,
                              номера, которые есть и в памяти, и в буфере, записываются в буфер
"Взять из памяти"           - список номеров из памяти копируется в буфер
"Очистить память"           - очистить память программы. Содержимое буфера не меняется.
"Таблица сравнений"         - формируется таблица сравнений номеров в памяти и в буфере
                              (Память, Буфер, Совпадающие, Нет в памяти, Нет в буфере)
--
"Исправление кодировки"     - у текста в буфере исправляется кодировка на Win-1251
